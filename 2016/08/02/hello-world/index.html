配置github和coding

注册github和coding账号（这就不多说）。
GitHub工程名必须是 【your_user_name.github.io】
coding工程名必须和你用户名一样,比如我的用户名cxl_leo,博客用户名也必须是cxl_leo.

接下来配置_config.yml文件

vim _config.yml

找到deploy

deploy:
type: git
repository:
github: https://github.com/chenxulun/chenxulun.github.io.git,master
coding: https://git.coding.net/cxl_leo/cxl_leo.git,master

  1. github: https://chenxulun.github.io/
