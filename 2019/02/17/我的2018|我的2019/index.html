我的2018，我的2019

2019.02.17 陈先生  热度 ℃

少年易老学难成，一寸光阴不可轻


——朱熹

我的2018

回顾我的2018，结果：都没有按照既定的目标前行，那就总结一下2018，送给文采不行，但是不断学习和挑战的自己。

关于学习

2018年首先献给自己，这一年，学习了很多，拓宽了知识边界，也学会游泳，同时收获了几个朋友。

  • 看书

看了近30本书，技术，历史，哲学各方面，收获最大的是享受这个过程，也意识到碎片化时间是最值得珍惜的时间。

  • 游泳&英语

印度出差之行补充了工作之外的不足，持续受益的不得不说：游泳和英语。

去印度之前 去印度之后
游泳 不会 300米来回20圈
英语 不敢开口，听不懂 可以沟通，电话沟通

关于技术

几点：

  • 相对惭愧，没有太大的进步，从面向用户端高并发的应用走向基于权限认证的后台系统，技术场景上hold的住。

  • 创业型公司，可以见证了从单体应用到拆服务到微服务整个演变过程，可以充分体会小而美的团队优点，从nginx到运维体系搭建到DB，以及整个技术架构，参与感很强，只要你愿意，每一个点都可以深入。

  • 进入相对大一点公司，成熟的监控体系，成熟的技术组织架构，同样可以让你学到你在小而美团队所得不到的。

  • 无论公司大小，技术或者架构终归是为了业务而服务的，能够快速支撑业务发展的架构就是最好的架构。

关于思考

两点：

  • 计划看《刻意练习》：重复性的事情做的时间越久，如果没有了思考和总结，会越来越狭窄，高人两项本领：洞察问题的能力，解决问题的能力。

  • 面对不公平的评价或者指责，学会保持沉默，有时候不解释是最好的解释方式；在发脾气的时候尽量慢半拍，思考一下，很多思想和从容态度都在此时发生质变。

我的2019

对于已经踏入的2019，给自己定几个小目标：

学习

  • 英语达到完全沟通水平，可以完全应对商务技术沟通(8月份之前)
  • 看书30本

生活

  • 游泳:规范化仰泳和蛙泳泳姿，锻炼身体

  • 交五个朋友

技术

  • 深入JVM优化

  • 学习Go语言，希望可以通过Go写一个简单的应用（12月之前）

努力是必须的
人类的全部智慧：等待和希望
